- `2026-05-24` `active` — **In an orchestrated yolo-ship worktree, I wrote a new file with a MAIN-checkout absolute path (`/home/.../ax-next/packages/...`) instead of my worktree path (`/home/.../ax-next/.claude/worktrees/<id>/packages/...`).** It landed as an untracked file in the main checkout — harmless this time (didn't clobber HEAD, git status showed it `??`), but vitest in the worktree then reported "No test files found" because the file wasn't there.

Palette Components, tokens, and microcopy Reference for every primitive used above.

Buttons — provider keys

Mono uppercase, ~11.5px, 0.08em tracking. Add key uses an outlined teal tint with a translateX hover hint; Edit key stays neutral so an already-configured row reads as “done.”

Buttons — provider key form

Same family — mono uppercase, ~11.5px. Save fills with accent teal; Retry is identical (label flips on error). Disabled state drops opacity to 0.5. Cancel is a hairline outline.

Buttons — model config

Larger CTA (12px, 0.1em tracking, 11×22 padding). Hover lifts 1px; disabled fills with --bg-deep and mute ink.

Status indicators

Configured Not configured

7px dot + 3px alpha glow. Teal for configured, ghost gray for empty. Label is mono uppercase 11.5px, 0.12em tracking.

Inputs

Mono input. Focus ring uses 3px alpha-accent halo; disabled drops to 0.6 opacity. Password is preferred so the value is masked while typing.

Native select (closed)

Custom-styled native <select>. The chevron is two 45°/135° gradients, no SVG. Browser owns the open-state UI when clicked. See Scene 07 for what a custom-combobox alternative could look like.
